# AI;DR

**AI didn't read.**

Drop `aidr.wtf` anywhere something was clearly written by AI and nobody read it.

## Usage

Post it as a tag, a reply, a caption — whatever fits:

- A 47-slide deck that's 90% ChatGPT filler → `aidr.wtf`
- A press release that reads like a prompt → `aidr.wtf`
- Terms of service nobody wrote and nobody will read → `aidr.wtf`
- That LinkedIn post → `aidr.wtf`

## That's it

No app. No account. Just a vibe and a domain.
